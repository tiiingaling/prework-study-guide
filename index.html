HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • !DOCTYPE html - defines document as html
  • head - contains information (metadata) to render the page, not visible to the user
  • body - content shown to the user, all visual and interactive elements must be placed inside this element
  • title - required element to define title for webpage, important for Search Engine Optimisation (SEO)
  • style – CSS styles
  • base - base URL
  • link - external resource
  • meta - metadata; character set, description, keywords, author, viewport etc
  • script - embeds client-side scripts
  • div - create division
  • span - create inline division
  • h(number) - indicates level of heading for text
  • p - paragraph/block of text
  • ul, ol, li - unordered lists, ordered lists, list items
  • img - image
  • Src – source
  • br - line break
  • a- anchor; creates links
  • Href – points to URL for link
  • Alt - text string that describes the appearance and functionality of the image
  • id - identify elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * applies to all visible elements
